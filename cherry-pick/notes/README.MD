# Cherry-Pick

## Cherry-pick with no conflicts

- Cherry-pick one commit-hash
  - Get the `commit-hash` of the commit you want to cherry-pick
    - `git cherry-pick commit-hash`
- Cherry-Pick more than one commit-hash
  - Get the `commit-hashes` of the commits you want to cherry-pick
    - `git cherry-pick commit-hash1 commit-hash2`
